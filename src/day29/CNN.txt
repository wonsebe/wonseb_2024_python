    - 레이어 에는 각 노드(뉴런)가 이전 레이어의 모든 노드와 연결된 상태 =완전 연결층
    - 기본 레이어 / 층 단계
        입력층(독립변수) ---> 은닉층1 ---> 은닉층2 ---> 은닉층3 ---> 출력층(종속변수)

[합성곱 신경망]: CNN
    - 이미지를 인식할 수 있는 능력을 갖춘 모델이다.
    - 사용처: 스마트폰 얼굴인식 잠금장치 , 자동차 자율주행
    - 핵심 개념 : 1.합성곱연산  2.폴링
    - CNN 구조:
        - 1. 입력층 : 학습할 원본 이미지들을 입력 받는다.
            32픽셀 * 32픽셀 크기의 이미지라면 2차원 리스트/배열로 입력받는다.

        - 2. 합성곱층: 이미지를 필터(커널)를 적용해서 중요한 패턴이나 특징을 추출한다. #이미지 패턴 감지 역할
            # 이미지에서 특징을 추출하는 합성곱 연산을 수행

        - 3. 폴링층 : 이미지 크기를 줄여 계산량을 줄이고, 중요하지 않는 세부 정보를 무시하고 모델의 성능을 높인다.
            # 영역의 크기를 조정한다.

        - 4. 완전 연결층 : 추출한 특징을 바탕으로 분류나 예측하기 위해 학습 한다.
            #모든 뉴런이 서로 연결된다.

        - 5. 출력층 : 최종 예측을 제공한다.
            #예측하고 확률을 출력한다.
    -CNN 특징
        - 1. 공감적 정보 유지
        - 2. 파라미터 효율성
        - 3. 자동 특성 추출
        - 4. 다양한 분야 적용

[합성곱]
    -CNN(합성곱 신경망)의 핵심 연산
    -이미지와 커널(필터)를 결합하여 중요한 특징을 추출하는 방식
    - 합성곱 원리
        -1. 커널(필터): 주로 작은 3x3 ,5x5 크기로 , 작은 영역(특정 부분)과 곱셈 및 덧셈 연산을 수행하여 새로운 값(특성맵)을 생성한다.
        -2. 스트라이드: 커널이 이미지 위해서 이동하는 간격을 의미한다. (전체 이미지에서 작은영역(특정부분)을 이동하면서 필터를 해야하므로 )
            #스트라이드가 1이면 1칸씩 이동 #스트라이드가 2이면 2칸씩 이동한다.  #스트라이드가 더 크면 빠른 처리가 가능하다. 세부정보가 누락될 수 있다. #적절히 사용
        -3. 패딩(css 패딩 그거 맞음) : 커널이 이미지 가장자리를 벗어나지 않도록 이미지를 둘러싼 테두리에 0으로 채워주는 기법 # 이미지 크기를 유지 하거나 특징 손실 방지한다.


[커널 (필터)]
    -커널은 보통 2차원(2D/2차원리스트)형태 가지고 , 주로 3x3, 또는 5x5 크기를 사용한다.
    -이미지의 특정 부분을 곱셈 및 덧셈을 통해 새로운 값(특성맵)을 생성한다.
    -커널 원리
        1. 합성곱 연산
        2. 특징 맵 생성: 각 특정한 특징(가장자리, 선,질감 등등) 을 나타낸다.
        3. 학습 가능한 파라미터(변수) : 초기에는 무작위로 설정되지만, 역전파 과정을 통해 손실(오차)을 최손화하는 방향으로 자동 조정된디.

[손실(오차)]
    -어떠한 물체나 현상을 측정한 값이 실제의 것과 동떨어진 정도
    -AI 중요한 부분: 진짜(실존한 값)와 가짜(컴퓨터가 학습해서 예측한 값) 의 차이를 줄이는 기술


[역전파]
    -신경망의 학습을 위한 핵심 기법(알고리즘) 이고 , 손실(오차) 함수의 기울기(그레디언트)를 계산하고 , 이를 통해 가중치 추출한다.

[가중치]
    - 표본(특정 집단)의 관측치에 대한 중요도를 나타내는 값
    - 예] 유튜브 알고리즘( 내가 많이 보는 유형의 영상을 추천한다, -축구분야 -나이 -여러가지 취향을 반영)
        -축구분야를 좋아하면 조금 더 축구분야의 가중치(점수)를 주고 추천한다.
    -예] 대학교 입시 내신성적 가중치(수학이 과학 점수 보다 더 많은 중요가 필요하면 수학 성적에 가중치를 더 부여해서 입학 점수 측정한다.)
        - 수학90, 과학100 일 때 수학 가중치가 80% , 과학 가중치가 20%
    - 신경망에서 중요한 파라미터 이고, 입력값에 가중치를 계산한 후 학습한다. 역전파를 통해 학습된 새로운 가중치를 다시 가중치를 업데이트 된다.
        # 이러한 작업을 반복적 학습으로 손실을 최소화 한다. #경사하강법 #수학 미분(아주 작은 변화를 계산)

    -전체적인 학습 흐름
        1. 데이터를 신경망에 입력한다.
        2. 입력한 값과 학습한 값 간의 차이를 계산하여 손실(차이)를 구한다.
        3. 역전파(계산법)를 통해 손실(차이) 함수에 대한 각 가중치의 기울기를 계산한다.
        4. 경사 하강법을 사용하여 가중치를 업데이트 한다.
        5. 이 과정을 반복하여 신경망의 성능을 최적화 한다.



[채널]
    -색상 정보를 담고 있는 각 구성 요소 이다.
    -빨강(R),파랑(B),초록(G) 세 가지 기본 색상 채널이 존재한다.
    -예]
        흑백 이미지: 1채널(회색)
        컬러 이미지: 3채널(R채널,G채널,B채널), RGBA(4채널)


[스트라이드]
    - 커널(필터)가 한 번에 몇 픽셀을 이동하는지 결정하며, 이미지의 크기와 특성 맵의 크기를 조정한다.
    - 주로 1 또는 2를 사용한다.
    - 높은 스트라이드는 계산 효율성 과 일반화에 좋고 세부정보손실이 우려된다.
    - 낮은 스트라이드는 세부정보 유지 하는데 계산비용 증가와 과적합이 우려된다.

[패딩]
    - 이미지의 가장자리에 픽셀을 추가하여, 일반적으로 0으로 채워진 픽셀이다.
    - 특징
        1. 특성 맵 크기 유지 : 패딩을 추가하면 필터가 이미지의 모든 부분을 고르게 처리 할 수 있다.
        2. 정보 손실 방지 : 종종 가장자리에 손실이 발생하는데 패딩을 추가하면 가장자리 정보가 손실 되는 것을 방지할 수 있다.
        3. 학습 안정성 향상 : 경계(가장자리)의 왜곡을 줄일 수 있다. 경계 효과를 줄인다.
        4. 연산 효율성 : 더 복잡한 패턴을 학습하는데 도움이 된다.

[폴링]
    - 필터(커널) 를 사용하여 이미지의 작은 영역에서 특정 통계 값을 계산하여 (연산)크기를 줄이는 과정이다.
    - 최대값이나 평균값 선택하여 그 영역의 하나의 값으로 대체한다.
    -특징
        1. 특징 추출: 중요한 특징을 추출하고 강조하며, 각 영역에서 큰 값 또는 평균값을 선택하므로 정보는 보존이 가능하다.
        2. 차원 축소: 메모리 사용량이 감소하며, 학습 속도 높이고 깊은 신경망을 설계할 수 있다.
        3. 불변성 향상: 더 일반화한 신경망을 학습 할 수 있다.
        4. 과적합 방지: 모델의 복잡성을 줄일 수 있다.

[픽셀]
    - 이미지를 구성하는 기본 단위 , 점. #729p/ 1000p

[이미지에서 커널(필터) 특성/특징 추출]
    - 픽셀 추출 [ 흑백이면 0~255 , 칼라이면 R(0~255) , G(0~255) ,B(0~255) ]
    - 주요 특성
        1. 모서리 / 선 : 1.사각형인지 삼각형인지
        2. 색상 / 질감 : 티셔츠의 매끈한 느낌 또는 거친느낌 인지
        3. 얼굴 인식 : 눈, 코, 입
        등등

    - 커널(필터) : 안경/에어컨필터 등등 처럼 시각적으로 이미지에서 중요한 특성(모서리, 질감)를 찾아내는 과정
    - 특성맵 : 이러한 찾은 특성을 기록
